\doxysubsection{Detailed Description}
An object that represent an user of the software. 

A user that has ID, a W\+SN topology and information, the trajectory area set (T\+AS). Use the above information, we can find the encoded trajectory using proposed algorithm in\+: \href{https://ieeexplore.ieee.org/abstract/document/8935061}{\texttt{ https\+://ieeexplore.\+ieee.\+org/abstract/document/8935061}}
